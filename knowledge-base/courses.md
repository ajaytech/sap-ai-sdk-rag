# Course Curriculum Documentation

## chunk_course_cs101
**Component:** CS101: Introduction to Computer Science
**Category:** Course Overview
**Content:**
This course provides a broad introduction to the fundamental concepts of computer science. It is designed for students with no prior programming experience and serves as the prerequisite for all advanced computing courses.

**Prerequisites:** None
**Learning Objectives:**
- Understand basic programming constructs (variables, loops, conditionals)
- Explain the difference between compiled and interpreted languages
- Write simple algorithms to solve mathematical problems

---

## chunk_topic_cs101_001
**Component:** CS101: Basic Programming Concepts
**Category:** Topic Detail
**Content:**
**Variables and Data Types:**
Students will learn about primitive data types including Integers, Floating-point numbers, Booleans, and Strings. Understanding how data is stored in memory is a key focus.

**Control Flow:**
- **If-Else Statements:** Conditional logic for decision making.
- **For and While Loops:** Iterative processes for repetitive tasks.

---

## chunk_course_wd201
**Component:** WD201: Modern Web Development
**Category:** Course Overview
**Content:**
This course covers the full spectrum of web development, from frontend layout to backend logic. Students will build responsive, interactive web applications using industry-standard tools and frameworks.

**Prerequisites:** CS101 or equivalent programming experience
**Learning Objectives:**
- Master HTML5 and CSS3 for structure and styling
- Develop dynamic interfaces with React.js
- Build RESTful APIs with Node.js and Express

---

## chunk_topic_wd201_001
**Component:** WD201: Frontend Architecture with React
**Category:** Topic Detail
**Content:**
**Component-Based Design:**
React allows developers to build complex UIs from small, isolated pieces of code called "components".
- **JSX:** A syntax extension that allows writing HTML-like code within JavaScript.
- **Props and State:** Mechanisms for passing data and managing internal component data.
- **Hooks:** Using `useState` and `useEffect` for state management and side effects.

---

## chunk_course_ds301
**Component:** DS301: Data Science and Analytics
**Category:** Course Overview
**Content:**
Explore the world of data-driven decision making. This course teaches students how to process large datasets, perform statistical analysis, and visualize results to tell a compelling story with data.

**Prerequisites:** CS101 and Basic Statistics
**Learning Objectives:**
- Clean and manipulate data using Python (Pandas/NumPy)
- Create insightful visualizations with Matplotlib and Seaborn
- Apply basic machine learning models for prediction

---

## chunk_topic_ds301_001
**Component:** DS301: Data Manipulation with Pandas
**Category:** Topic Detail
**Content:**
**DataFrames and Series:**
Pandas is the primary library for data manipulation in Python.
- **DataFrames:** 2D labeled data structures (like tables).
- **Indexing and Selecting:** How to retrieve specific rows or columns.
- **Groupby Operations:** Aggregating data based on specific criteria.

---

## chunk_course_cy401
**Component:** CY401: Cybersecurity Fundamentals
**Category:** Course Overview
**Content:**
Understand the principles of securing digital assets. This course covers the threat landscape, defensive strategies, and the legal/ethical considerations of cybersecurity.

**Prerequisites:** CS101 and Networking Basics
**Learning Objectives:**
- Identify common cyber threats (Phishing, Malware, SQLi)
- Understand encryption and hashing techniques
- Implement secure authentication mechanisms

---

## chunk_topic_cy401_001
**Component:** CY401: Web Security and Vulnerabilities
**Category:** Topic Detail
**Content:**
**The OWASP Top 10:**
A standard awareness document for developers and web application security.
- **Injection:** SQL, NoSQL, and OS injection.
- **Broken Authentication:** Protecting session tokens and credentials.
- **Cross-Site Scripting (XSS):** Preventing malicious scripts from executing in a user's browser.
